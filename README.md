
# automacao-cadastro-produtos-python
Projeto desenvolvido utilizando Python, PyAutoGUI e Pandas para automatizar o cadastro de produtos em um sistema web.

🚀 Tecnologias

* Python
* Pandas
* PyAutoGUI

✨ Funcionalidades

* Login automático
* Leitura de CSV
* Cadastro automático
* Navegação automática entre campos

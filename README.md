
# Automação em Python para cadastro de produtos em sistema web utilizando PyAutoGUI, Pandas e leitura de arquivos CSV.
* Projeto desenvolvido utilizando Python, PyAutoGUI e Pandas para automatizar o cadastro de produtos em um sistema web.

## Objetivo

Este projeto automatiza tarefas repetitivas de cadastro de produtos, reduzindo o tempo gasto com digitação manual e minimizando erros.


🚀 Tecnologias

* Python
* Pandas
* PyAutoGUI

✨ Funcionalidades

* Login automático
* Leitura de CSV
* Cadastro automático
* Navegação automática entre campos
